# Personal-Diary
A cute and secure personal diary made with python and tkinter.
You can write your daily thoughts,experiences and plans.You can edit, remove and personalize the file name.

## Features
** Password Protection **
Opens only with your secret password. 

** Create New Entries **
Create and write your thoughts with custom file name.

** Edit Entries **
In case you like to edit the entries.

** Remove Entries **
Remove the unnecessary entries.

** Cozy GUI **
The cozy vibe every diary needs.


## How It Works

1. At first the login window appears, you enter the secret password and you are directed towards main window.
2. Main window consists of text area in which you write your thoughts and save entry.
3. Then you can set custom name for your entry.
4. You can edit and delete the existing entries.
5. Your entries are properly stored.
